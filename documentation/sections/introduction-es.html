Los datos de empleo son fundamentales para el análisis socioeconómico, la planificación territorial y la evaluación de políticas públicas. Sin embargo, con frecuencia se publican en formatos heterogéneos y con escasa semántica explícita, lo que dificulta su interoperabilidad y reutilización. En este contexto, el vocabulario W3C Data Cube proporciona un marco adecuado para representar datos estadísticos multidimensionales de forma estructurada y reutilizable. Además, la reutilización de vocabularios y listas de códigos consolidadas permite describir de manera más precisa dimensiones relevantes como el territorio, el periodo, el sexo, la edad o la actividad económica. El objetivo de este vocabulario es definir la estructura semántica del cubo de datos de empleo, especificando las dimensiones y medidas necesarias para representar observaciones estadísticas de empleo de forma interoperable y conforme a estándares abiertos.
